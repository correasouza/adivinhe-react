## Objetivo

Desenvolva uma interface de um jogo de adivinhação de palavras interativo conforme o design do Figma.

## Tecnologias a Serem Utilizadas

- React.js
- TypeScript
- Componentes e Propriedades
- Hooks (useState e useEffect)
- CSS Global e CSS Modules

## Funcionalidades do Jogo

- Deve sortear uma palavra
- Deve permitir o usuário inserir uma letra de palpite
- Deve ter um histórico das letras utilizadas
- Letras com palpite correto ficam verdes, e as erradas ficam amarelas
- Validar as letras para não permitir a entrada da mesma letra
- Botão para reiniciar o jogo
